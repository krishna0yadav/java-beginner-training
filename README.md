# Java Beginner Training

A collection of beginner-level Java programs and learning materials from college training (2020).

## Project Structure

- **JavaRefresher/**: Contains refresher programs on core Java concepts
  - `equalscomparison/`: Demonstrates equals(), hashCode(), and Comparable interface
  - `lambdas/`: Lambda expressions and functional interfaces examples
  
- **training/**: Class notes and training materials (text files)
- **junitproject/**: Unit testing examples using JUnit

## Topics Covered

- Object equality and comparison
- Lambda expressions
- Collections and Streams
- JUnit testing

## Prerequisites

- Java 8 or higher
- JUnit 4 (for junitproject)

## Getting Started

Each module can be compiled and run independently. Ensure your IDE (Eclipse, IntelliJ) is properly configured with Java path.
