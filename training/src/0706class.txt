SQL Languages:

DDL: Data Definition Language
[Tables]
	-> create : to Create new Table in the database.
	-> alter  :
		-> add one column: ALTER TABLE table-name ADD col-name with definition
		-> add multi cols: 
				ALTER TABLE table-name 
				ADD(
				col-name with definition,
				col-name with definition,
				col-name with definition
				);
		-> modify:
				ALTER TABLE table-name
				MODIFY column-name and its definition
		-> Drop:
				ALTER TABLE table-name
				DROP col-name;
		-> rename:
				ALTER TABLE table-name
				RENAME COLUMN old-col-name TO new-col-name;
				
	-> drop	  : DROP TABLE table-name;
	-> rename : RENAME old_table-name to new_table_name;
	-> truncate : TRUNCATE TABLE table-name;
