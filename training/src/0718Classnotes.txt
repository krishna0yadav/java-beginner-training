Formatting Tags : marquee
------------------------------
<marquee></marquee>

Attributes:
	direction: left,right,up,down
	bgcolor: 
	width: percentage
	height: percentage
	behavior: alternate,slide
	scrolldelay: scrolling speed

Example:
<html>
	<head>
		<title>My first HTML Program</title>
	</head>
	
	<body>
		<marquee direction="left" bgcolor="blue" behavior="scroll" scrolldelay="500">
		My first HTML Program
		</marquee>
	</body>
</html>

List
----
1. Ordered List
	<ol></ol>
2. Unordered List
	<ul></ul>
3. Definition List
	<dl><dl>

In ordered and Unordered case, each list items start with <li> tag.
In Definition list, item start with <dt> and <dd> tags.

1. Ordered List
	<ol></ol>
	
	Attributes:
	start: value
	type: a,A,1,i,I
	
	Example:
	<ol start="10" type="A">
			<li>One</li>
			<li>Two</li>
			<li>Three</li>
			<li>Four</li>
			<li>Five</li>
		</ol>

2. Unordered List
	<ul></ul> 
	
	Attributes:
	type: circle,square,disc,none
	
	Example:
	<ul type="square">
			<li>One</li>
			<li>Two</li>
			<li>Three</li>
			<li>Four</li>
			<li>Five</li>
		</ul>

3. Definition List
	<dl><dl>
	
	Example:
	<dl>
			<dt>Introduction</dt>
			<dd>My first HTML Program</dd>
		</dl>

Form tag
--------
<form></form>

	attribute
	
	name:
	method:
	action:
	
	Example:
	<form name="registration" method="GET" action="serverside filename" >
	</form>

GET method:
-> It is used to forward the request.
-> It is restricted to send upto 1024 character only.
-> It produce a string that appears in Address bar
-> Never use GET method if you have sensitive information to sent to the server.
-> GET cannot be used to send binary data.

POST method:
-> It transfers the information using HTTP header.
-> doesn't have any restriction on data size.
-> can be used to send binary data.
-> The data sent by POST method goes through HTTp header so security depends on HTTP protocol.
