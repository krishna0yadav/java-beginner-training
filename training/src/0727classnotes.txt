Regular Expression
--------------------
-> It is also called Regex.
-> It is used to perform pattern matching

---------------
Extra:
	refrence website: w3school
---------------

pattern syntax:
			/pattern/

Example:
	
<html>
<head>
	<script>
		function show()
		{
			var a = document.getElementById('user').value;
			var pattern=/hello/;
			if(a.match(pattern))
			{
				document.getElementById('sp1').innerHTML = "Valid";
			}
			else
			{
				document.getElementById('sp1').innerHTML = "Invalid";
			}
		}
	</script>
</head>
<body>
<form>
	<input type="text" id="user" />
	<input type="button" value="show" onclick="show()"/>
</form>
	<span id="sp1"></span>
</body>
</html>


---------------------------------------------
^	:	pattern start with ^ to match string at begining
Example:
<script>
		function show()

			var pattern=/^hello/;

			}
			else
			{
				document.getElementById('sp1').innerHTML = "Invalid";
			}
		}
	</script>

Meta characters
----------------
\d	: matches any digit, equivalent to [0-9]
\D 	: matches any character that's not a digit , ET  [^0-9]
\w  : matches any aplhanummeric character , [A-Za-z0-9_]
\W  : matches any non alpanumeric charactes, [^A-Za-z0-9_]

quantifiers
-----------
+	: match one or more items
*   : match zero ore more items
{n} : match exactly n items
{n,m}: match between n and m items.
?    : to make item optional, zero or one

Example:
<script>
		function show()
		{
			var a = document.getElementById('user').value;
			var pattern=/^\d+$/;
			if(a.match(pattern))
			{
				document.getElementById('sp1').innerHTML = "Valid";
			}
			else
			{
				document.getElementById('sp1').innerHTML = "Invalid";
			}
		}
	</script>
Example:
