Bootstrap Grid System
----------------------
-> It is an easy way to create web page layout.
-> Upto 12 columns across the page.

Grid classes:
	lg	:	for large desktop
	md  :   for desktop/laptop
	sm  :   for tablets
	xs  :   for mobile phone

How to create grid:
1. create "container"
2. create "row"
3. create "columns" inside the row(step-2).

Example:
<html>
<head>
	<link rel="stylesheet" type="text/css" href="E:/Software/bootstrap-3.4.1-dist/css/bootstrap.min.css" />
</head>
<body>
	<div class="container">
		<div class="row">
			<div class="col-sm-7">
			
			</div>
			<div class="col-sm-1">
				
			</div>
			<div class="col-sm-4">
				
			</div>
		</div>
	</div>
</body>
</html>
